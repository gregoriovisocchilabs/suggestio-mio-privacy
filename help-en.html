About the App

Suggestio Mio is a smart decision-making and random picker app. Create sets with any options — meals, movies, activities, challenges — and let the app draw a result using a spectacular animation. Perfect when you have too many options and can't make up your mind!

No registration, no cloud. All your data is stored exclusively on your device. You can create sets and draw results offline.

How to Use
1
Create a new set
Tap "+" on the main screen or use a ready-made template. Give your set a name and choose a color.
2
Add options
Type options manually or use voice input (microphone icon). At least 2 options are required.
3
Choose mode and animation
Select draw mode (with/without repetition) and animation style. Set duration and cycle count.
4
Tap DRAW
Press the large DRAW button and watch the animation. The result is shown when the animation ends.
5
Confirm or redraw
If confirmation is enabled, you can accept the result or draw again.
6
Check history and status
The history icon shows all past draws. The options status screen shows which have been drawn already.

Draw Modes
🔁
With Repetition
Each draw is independent. Any option can appear multiple times.

Smart lock – same option won't appear three times in a row if another is available.
✅
Without Repetition
Each option can only be drawn once. After the pool is exhausted you can reset it.

The status screen shows drawn and remaining options.

📤
Export
In Settings tap "Export Sets". A JSON file will be saved or shared via your file/drive/email app.

Tap "Import Sets" and select a previously exported JSON file. Sets will be added to the app.

Privacy & Data

✅ What the app does with your data

  • All sets, options, and history are stored exclusively on your device (SQLite).
  • The app does not collect or transmit any personal data to external servers.
  • No user account, no login, no cloud.

📢 Ads (Google AdMob)

  • The app shows Google AdMob ads to fund development.
  • On first launch you'll be asked to consent to personalised or non-personalised ads.
  • You can change your choice in Settings → "Manage Ad Consent".
